Instructions for using the Back-end

Using it first time:
 Make sure the following bullet-points are installed and function:
  -  Docker
  -  Postman
  -  TablePlus

Starting the Back-End:
  - Type into the terminal "dcoker compose up" for starting docker
  - Start the application via the start-button
  - Make sure that the port for doceker and the back-end are not used 

Using Postman:
  - Make sure that Postman ist intalled and function
  - Start Postman on the local machine (Desktop)
  - Open Postman on the Web, there you can test the endpoints with the following adress:
    "http://localhost:8080/api/<INSERT ENDPOINT HERE>"
    example: "http://localhost:8080/api/corgis"
